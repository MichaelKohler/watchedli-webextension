By default the 'More information' link points to Google for a Google search with the specified episode. You can overwrite the URL here. Please make sure to put a placeholder '%S' into the link. The '%S' part will be replaced by the series name and the episode number automatically. You can do a search on any website you would like to link and replace the search term with '%S' before copying it here.
Example: https://duckduckgo.com/?q=%S
